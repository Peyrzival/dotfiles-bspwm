### Programs Needed
* bspwm
* sxhkd 
* rofi
* polybar
* feh 
* dunst
* picom 
* scrot 

### Important
* Fonts needed: JetBrainsMono Nerd Font (polybar) and SpaceMono Nerd Font (alacritty).
* Changes some values in polybar like wlan for your specific wifi adapter, battery module, etc.
